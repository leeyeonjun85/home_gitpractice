## MDN 'Web과 함께 시작하기'

### MDN '기본 소프트웨어 설치하기'
- 코드 편집기
  - 코드 작성을 위한 편집프로그램으로는 Sublime, Brackets, Atom, Visual Studio Code 등이 있다.
  - 나는 기존에 사용하던 [Visual Studio Code](https://code.visualstudio.com/)를 사용할 것이다.
- 웹 브라우저
    - 기존에 사용하던 [Chrome](https://www.google.com/chrome/)을 사용할 것이다.
